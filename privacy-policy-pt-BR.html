Política de Privacidade

CostKeeper

Introdução
Bem-vindo ao CostKeeper (a seguir, «este Aplicativo»). Entendemos a importância das informações pessoais para você e cumpriremos os requisitos legais para proteger seus dados pessoais com medidas de segurança adequadas.
1. Informações que coletamos
1.1 Dados armazenados localmente
Este Aplicativo utiliza armazenamento local. Todos os seus dados (incluindo nomes de itens, preços, categorias, data de compra, data de validade, notas, imagens, etc.) são armazenados apenas no seu dispositivo e não são enviados para nenhum servidor ou terceiro.
1.2 Informações que não coletamos
Este Aplicativo não coleta, armazena nem transmite nenhuma de suas informações pessoais, incluindo, mas não se limitando a:
  • Nome, número de telefone, endereço de e-mail ou outros dados de contato
  • Números de identidade, números de cartão bancário ou outras informações sensíveis
  • Informações de localização ou informações do dispositivo
  • Histórico de navegação ou dados de comportamento de uso
2. Como usamos as informações
Os dados armazenados localmente no seu dispositivo são usados apenas para os seguintes fins:
  • Registrar e gerenciar as informações dos seus itens
  • Calcular o valor diário e a duração de uso dos itens
  • Fornecer estatísticas por categoria e funções de análise de dados
  • Suportar a funcionalidade de exibição de imagens
3. Compartilhamento e transferência de informações
Como este Aplicativo utiliza uma arquitetura de armazenamento puramente local, nós não:
  • Compartilhamos nem transferimos seus dados a terceiros
  • Utilizamos seus dados para fins comerciais
  • Transferimos seus dados para fora da sua região
4. Armazenamento e proteção de dados
4.1 Local de armazenamento dos dados
Todos os seus dados são armazenados no armazenamento local do seu dispositivo.
4.2 Responsabilidade pela segurança dos dados
Observe que, como os dados são armazenados inteiramente de forma local:
  • Desinstalar o Aplicativo resultará em perda de dados
  • Limpar o cache do Aplicativo resultará em perda de dados
  • Os dados não serão sincronizados automaticamente ao trocar de dispositivo
5. Seus direitos de gerenciar dados
Você tem controle total sobre seus dados:
  • Você pode adicionar, editar ou excluir qualquer registro de item a qualquer momento
  • Você pode desinstalar o Aplicativo a qualquer momento para excluir todos os dados
  • Você pode escolher livremente se deseja usar a funcionalidade de imagem
6. Uso de permissões
Este Aplicativo pode requerer as seguintes permissões do sistema:
  • Permissão de câmera: Para tirar fotos dos itens (uso local apenas)
  • Permissão de galeria de fotos: Para selecionar ou salvar imagens de itens (uso local apenas)
Você pode gerenciar essas permissões a qualquer momento nas configurações do seu dispositivo.
7. Privacidade de crianças
Este Aplicativo é destinado a todas as faixas etárias e não envolve a coleta de informações pessoais de crianças. Se menores tiverem dúvidas ao usar este Aplicativo, recomendamos consultar um responsável.
8. Alterações na Política de Privacidade
Podemos atualizar esta Política de Privacidade periodicamente. Quando houver alterações significativas nesta política, notificaremos os usuários por meio de notificações no Aplicativo ou outros meios razoáveis.
